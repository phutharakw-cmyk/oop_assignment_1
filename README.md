# Library Management System (OOP)

This project converts a procedural library management program into an object-oriented design.

## Classes Used
- **Book**: Stores book information and manages copy availability.
- **Member**: Stores member information and tracks borrowed books.
- **Library**: Manages collections of books and members, and handles borrow/return operations.
